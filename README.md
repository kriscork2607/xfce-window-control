# XFCE Window Control

Per-monitor window tiling and monitor switching for XFCE/X11.

## Shortcuts

Super + NumPad:

7 8 9
4 5 6
1 2 3

## Mapping

Super + KP_7 = top-left  
Super + KP_8 = top  
Super + KP_9 = top-right  

Super + KP_4 = left half  
Super + KP_5 = full current monitor  
Super + KP_6 = right half  

Super + KP_1 = bottom-left  
Super + KP_2 = bottom  
Super + KP_3 = bottom-right  

## Move window between monitors

Super + Left = move left  
Super + Right = move right  
Super + Up = move up  
Super + Down = move down  

## Files

tile-active-monitor.sh  
move-active-monitor.sh  

## Install

Copy scripts to ~/bin/ and make them executable.
